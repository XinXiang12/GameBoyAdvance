# GameBoyAdvance
JetFighter
Xin Xiang - CS2110 HW08
A very exciting game. It's similar to jet fighter. 

Press enter (RETURN on computer keyboard) to start the game.
On the play screen, use arrow buttons to move your jet, press A (Z on computer keyboard) to shoot all the enemies out. You have 10 bullets total, if you hit one enemy or run out of bullet, you will DIE.

Press SELECT (DELETE on computer keyboard) to reset the game at anytime

Have Fun!
